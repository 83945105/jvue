# 更新日志

  • v1.0.3
    • 新增
      • 新增页面，控件类型列表头添加快捷全部设置为 详情 功能
      • 插槽数据类型支持设置为空
      • 生成代码支持简化模式
      • 生成代码页面支持一键复制代码
    • 优化
      • 下拉树设置异步加载方法后不再生成tree-data相关代码
      • 新增功能，控件类型可以选择 描述 控件了
    • 修复
      • 下拉树在表单中初始化会触发表单校验问题
      • 对象数组表单生成代码部分属性丢失问题
      • 文件上传开启拖拽上传和照片墙后，生成代码界面预览样式异常问题
      • 新增页面保存功能失效问题
      • 表单项配置，控件类型，选中描述展示为单行文本问题
